Title: Hack X
Prerequisites: Arrays, xor
Time Complexity: O(n)
Method Used: Brute Force
Instructions:
We can follow the instructions and get the answer directly.

  1. We will iterate through every element in the array and find the xor of each element and store it in a variable xr.
  2. Along with that, we add the elements to get the sum of the array and store it into a variable sum.
  3. We use an if statement to check if 2xr is equal to the sum as sum/xr=2 and if it is we print -1 which means the array already satisfies the condition else check if xr=0 if it is then print the sum as that would be the missing number, else print the two numbers which are missing a and b. Here a would be the xr and b would be xr+sum as:
    A xor B=C then C+A+B is always equal to 2    C.
    Eg: 2 xor 6=4 and 4+6+2=16 which is 2*(6+2).
